# FlatinyEngine
2D engine made in OpenTK/C# Inspired by Unity,Monogame and GMS2
## Sub-modules
- Graphics<br/>
Core of rendering, include:
- - Mesh rendering
- - Shaders
- - Textures
- - Fonts
- GUI<br/>
Graphical User Interface, include:
- - Canvas
- - Empty rect
- - Image rendering
- - Text rendering
- ProcessManagment<br/>
Combines all modules into one system
- SceneManagment<br/>
Provides access and control of the game scene
- Components<br/>
Components of game objects,include:
- - Component(Base)
- - MeshRenderer
- - SpriteRenderer
- - TextRenderer
- - Transform
- Console logger<br/>
Logger for debug 
- Input<br/>
Handle input
- Mathf<br/>
Useful functions
